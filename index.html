Flip-slider

Flip Slider Plugin with CSS 3, Pure JavaScript

Download .zip Download .tar.gz View on GitHub

Introduction

This is a slider with Flip effect animation. I used CSS3 to create this with pure javascript. No any JS library like jQuery is used, so this is very light and fast.

Features

  • Stylish Flip Animation
  • Pure JavaScript, no jQuery or any other JS library used
  • Very light file size. (3kb uncompressed version JS)
  • Supports Images and HTML contents both
  • Highly Customizable
  • Readable CSS and JS, well commented so that you can make your changes as per your requirements

How to use?

Set up your HTML markup as example and create an instance of the plugin class.

var myFlip = new flipSlider({ container: document.querySelector(".flip-slider") });
